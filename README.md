Add recipes for vanilla items that are not meant to be crafted!

In order to use this mod, you need Minecraft version 1.20.1 and Forge (47.4.0 is recommended)

## Current recipes:
- Crying Obsidian
- Diamond Horse Armor
- Enchanted Golden Apple
- Gilded Blackstone
- Golden Horse Armor
- Iron Horse Armor
- Name Tag
- Nautilus Shell
- Saddle
- Slimeball
- Trident

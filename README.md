Add recipes for vanilla items that are not meant to be crafted!

In order to use this mod, you need Minecraft version 1.20.1 and Forge (47.4.0 is recommended)

## Current recipes:
- Bell
- Brown Mushroom Block
- Cobweb
- Crying Obsidian
- Chainmail Armor
- Diamond Horse Armor
- Enchanted Golden Apple
- Gilded Blackstone
- Golden Horse Armor
- Iron Horse Armor
- Large Fern
- Mushroom Stem
- Name Tag
- Nautilus Shell
- Red Mushroom Block
- Saddle
- Slimeball
- Tall Grass
- Totem of Undying
- Trident
